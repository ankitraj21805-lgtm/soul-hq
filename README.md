# SOUL HQ

A clean full-stack responsive website for SOUL Syndicate with public pages, member sections, joining workflow, events, alliance information, gallery, rules, login, and admin-focused routes.

## Overview

SOUL HQ is a community/clan website project designed to present a professional group identity, onboarding flow, event information, and admin-managed content sections. It demonstrates full-stack product planning with frontend routes, Supabase integration, authentication direction, and deployment-ready environment configuration.
