# SOUL HQ

Clean full-stack responsive website for SOUL Syndicate.

Routes: Home, About, Members, Join, Events, Alliance, Gallery, Rules, Login, Admin.

Required Vercel environment variables:

NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY
SUPABASE_SERVICE_ROLE_KEY
NEXT_PUBLIC_SITE_URL

Run schema in Supabase from `supabase/schema.sql`.
